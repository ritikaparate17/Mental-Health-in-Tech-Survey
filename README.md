# Mental Health Analysis of IT Professionals

## Project Overview
An interactive Streamlit dashboard that analyzes mental health trends among IT professionals using survey data.

## Features
- Mental health insights dashboard
- Interactive visualizations
- Depression analysis
- Promotion and workplace relationship analysis
- Correlation heatmaps
- Demographic analysis

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
